Selector/Combinator Description Example Targets

ID Selector Selects an element with a specific ID. #myId Element with id="myId"

Pseudo-classes

Pseudo-class Description Example Targets
:hover Selects and styles an element when hovered over. a:hover The <a> element when hovered over
:active Selects and styles an element when activated. button:active The <button> element when activated
:focus Selects and styles an element when focused. input:focus The <input> element when focused
:first-child Selects the first child of a parent. p:first-child The first <p> element within its parent
:last-child Selects the last child of a parent. li:last-child The last <li> element within its parent
:nth-child(n) Selects the nth child of a parent. div:nth-child(2) The second <div> element within its parent
... (and a smorgasbord of other pseudo-classes)

Pseudo-elements

Pseudo-element Description Example Targets
::before Inserts content before the content of an element. p::before Inserts content before each <p> element
::after Inserts content after the content of an element. div::after Inserts content after each <div> element
::first-line Selects the first line of text in an element. p::first-line The first line of text in each <p> element
::first-letter Selects the first letter of text in an element. h1::first-letter The first letter of each <h1> element

Attribute Selectors

Attribute Selector Description Example Targets
[attribute^=value] Selects elements with an attribute that starts with a specific value. a[href^="https://"] All <a> elements with an href starting with "https://"
[attribute$=value] Selects elements with an attribute that ends with a specific value. img[src$=".jpg"] All <img> elements with a src ending with ".jpg"
[attribute*=value] Selects elements with an attribute that contains a specific value. input[type*="text"] All <input> elements with a type containing "text"
